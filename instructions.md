# Instructions

- Imagine you are a mid-level software developer with experience with web applications
- Your task will be to create a web game with backend in NodeJS and frontend in React that utilizes mobile phone as a tool in playing

## Game Description

- The game will be the famous card game "Uno!"
- It will be played between up to 4 players in an online lobby
- I want the players to see the playing table (where the played cards and the deck are) on their PC screens, but their own hand will be shown on their phone
- The act of dragging a card to the top edge of the phone will result in the card being played

## Required functionalities

- The game must have authentication and authorization of the players
- The game must utilize websockets to some extent
- Use Typescript
- Use Material UI for the UI of the frontend

## Additional information

- If you have any questions or if you are not sure about the task, please ask, don't guess
- Create separate NPM project for the frontend and the backend.
